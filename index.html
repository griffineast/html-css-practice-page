HTML

HTML stands for Hyper Text Markup Language. It is used to create the structure and content of web pages. HTML consists of a series of tags that define different elements such as headings, paragraphs, images, links, forms, and more.

HTML tags are enclosed in angle brackets, like this: <tagname>content</tagname>. Some common HTML tags include:

Tag Usage
<h1> - <h6> for headings
<p> for paragraphs
<a> for links
<img> for images
<form> for forms

CSS

CSS stands for Cascading Style Sheets. It is used to add style and layout to web pages. CSS consists of a series of rules that define how HTML elements should be displayed, such as colors, fonts, sizes, positions, and more.

CSS rules consist of a selector and a declaration block. The selector specifies which HTML elements should be styled, and the declaration block contains one or more property-value pairs, like this:

              selector {
                property1: value1;
                property2: value2;
                /* more properties... */
              }

Some common CSS properties include:

Property Usage
color sets the text color
background-color sets the background color
font-family sets the font family
font-size sets the font size
text-align aligns the text
border adds a border to an element
margin sets the margin space around an element
padding sets the padding space inside an element
